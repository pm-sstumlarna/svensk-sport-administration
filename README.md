# svensk-sport-administration
Detta är ett projekt som jag påbörjar för att fler skall kunna kunna bidra till att göra svensk idrott ännu bättre. Det finns många ledare, tränare och föräldrar som engagerar sig ideellt så då känns det som en självklarhet att även bjuda in dem som arbetar med mjukvara. Tillsammans kan vi göra svensk idrott ännu bättre.
