# Svensk Sport Administration

## Projektets Syfte

Detta är ett projekt som jag påbörjar för att fler skall kunna bidra till att göra svensk idrott ännu bättre. Det finns
många ledare, tränare och föräldrar som engagerar sig ideellt så då känns det som en självklarhet att även bjuda in dem
som arbetar med mjukvara. Tillsammans kan vi göra svensk idrott ännu bättre.

## Projektstyrning

### Bidra med kod

Vi är måna om att utvecklingen i detta projekt sker på ett säkert,
effektivt och kontrollerat sätt. Har du funktionalitet som du vill bidra med så kan du skicka en förfrågan om att
inkludera din kod i huvudprojektet. När den kommer in så granskas koden så att den följer projektets riktlinjer vad
gäller kodstil, testning och dokumentation.

### Bidra med testning

Testning är en viktig del av utvecklingen för att kunna säkerställa en stabil och säker användarupplevelse för alla
intressenter. Vill du vara med och ta fram en strategisk plan för testning så kontakta mig

### Bidra med dokumentation

Att dokumentationen är bra och uppdaterad är viktigt. Om du har förslag på hur vi kan förbättra dokumentationen
eller har dokumentation som du vill bidra med så tar vi gärna emot den hjälpen.

### Bidra ekonomiskt

Har du intresse av att stödja projektet ekonomiskt så tar vi gärna emot din donation. Din donation kommer att användas
för att finansiera resor så att vi kan träffa intressenter, utbildning för dem som bidrar till projektet och eventuella
kostnader knutna till utvecklingen av projektet.
